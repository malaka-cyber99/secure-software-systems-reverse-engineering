# Reverse Engineering and Runtime Manipulation of Legacy Game Software

## Overview
This project was completed for the Secure Software Systems module at SLIIT.

The objective was to analyze a legacy game executable using reverse engineering tools and modify assembly instructions during runtime.

## Tools Used
- Ghidra
- x32dbg
- Cheat Engine
- Windows 11

## Tasks Completed
- Static analysis of executable
- Dynamic analysis during gameplay
- Memory and register identification
- Assembly-level modification
- Security vulnerability analysis
- Mitigation strategies

## Key Concepts
- Reverse Engineering
- Assembly Language
- Runtime Debugging
- Memory Manipulation
- Anti-Debugging
- Integrity Checking

## Academic Purpose
This project was completed strictly for educational and research purposes.
